# 小峰出題技能

## 觸發條件
- 「小峰出題」
- 「出 [科目] 的題目」
- 「生成 [主題] 測驗」

## 執行步驟

### 步驟 1：確認範圍
詢問老師：
- 科目 / 單元
- 難度（國小/國中/高中）
- 題型（選擇/填充/問答）
- 題數

### 步驟 2：生成試題
生成格式：
```
題目 1：[題幹]
(A) [選項]  (B) [選項]  (C) [選項]  (D) [選項]
```

### 步驟 3（可選）：輸出為 HTML 簡報
```bash
python tools/slide.py --title "[科目] 隨堂測驗" --content "[題目內容]" --theme light
```

### 步驟 4（可選）：語音唸題
```bash
python tools/voice.py "請同學作答第一題：[題幹]"
```

## 注意事項
- 不直接給出答案，除非老師明確要求「答案版」
- 題目要配合該年級課綱
- 如需圖片輔助，搭配 image_gen.py 生成情境圖
